Class ImageInfo

  • java.lang.Object
    • org.apache.commons.imaging.ImageInfo
  • Direct Known Subclasses:
    PngImageInfo

    public class ImageInfo
extends Object
    ImageInfo represents a collection of basic properties of an image, such as width, height, format, bit depth, etc.
